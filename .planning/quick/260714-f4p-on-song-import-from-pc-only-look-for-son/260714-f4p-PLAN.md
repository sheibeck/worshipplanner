<objective>
On PC song import, let the user skip songs that have already been imported by adding an
"Import new songs only" checkbox to the import modal. When on (the default), only songs
that are not already in the library are upserted; already-imported songs are skipped.

Purpose: Avoid re-processing/re-writing the entire PC library on every import when the
user only wants to pull in newly-added songs.
Output: Exported matching helper + unit test, and a wired checkbox in PcImportModal.
</objective>
