| viewer (non-editor) → /services/:id Roles tab | `/services/:id` has no requiresEditor guard, so a viewer can reach this view; the Roles tab must not read editor-only roles/quarters/people. |
| editor → override control → services store | Editor selects override people; writes go through the scoped-dot-path store actions (17-03). |
