    5. Call `authStore.setPcCredentials(pcAppIdInput.value, pcSecretInput.value, authStore.pcServiceTypeId)`

    9. Wrap all in try/catch — on error, set pcValidationError = error message
    10. Set pcValidating = false in finally block
