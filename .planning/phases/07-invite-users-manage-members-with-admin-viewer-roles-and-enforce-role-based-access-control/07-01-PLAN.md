**A. Rewrite `firestore.rules`:**

1. **Replace `isOrgAdmin` with `isOrgEditor`:**
   ```
   function isOrgEditor(orgId) {
     return isSignedIn() &&
       get(/databases/$(database)/documents/organizations/$(orgId)/members/$(request.auth.uid)).data.role == 'editor';
   }
   ```
   Keep `isOrgMember(orgId)` unchanged (existence check only).

2. **Update organization-level rules:**
   - `allow read: if isOrgMember(orgId);` (unchanged)
   - `allow write: if isOrgEditor(orgId);` (was isOrgAdmin)
   - `allow create:` rule unchanged

3. **Update members subcollection:**
   - `allow read: if isOrgMember(orgId);`
   - `allow write: if isOrgEditor(orgId);` (was isOrgAdmin)
   - `allow create: if isSignedIn() && request.auth.uid == uid;` (unchanged — needed for self-registration and invite acceptance)

4. **Add invites subcollection rules:**
   ```
   match /invites/{email} {
     allow read: if isOrgEditor(orgId);
     allow write: if isOrgEditor(orgId);
     allow delete: if isOrgEditor(orgId) || (isSignedIn() && request.auth.token.email.lower() == email);
   }
   ```

5. **Replace the wildcard `/{collection}/{docId}` catch-all with explicit collection rules:**
   ```
   // Services — viewers can read, only editors write
   match /services/{docId} {
     allow read: if isOrgMember(orgId);
     allow write: if isOrgEditor(orgId);
   }

   // Songs — editors only (viewers cannot read songs)
   match /songs/{docId} {
     allow read, write: if isOrgEditor(orgId);
   }

   // All other nested collections — editors only
   match /{collection}/{docId} {
     allow read, write: if isOrgEditor(orgId);
   }
   ```
   Note: Firestore matches the most specific rule first, so /services and /songs rules take precedence over the wildcard.

6. **Add inviteLookup rules:**
   ```
   match /inviteLookup/{email} {
     allow read: if isSignedIn() && request.auth.token.email.lower() == email;
     allow create: if isSignedIn();
     allow delete: if isSignedIn() && request.auth.token.email.lower() == email;
   }
   ```

7. **shareTokens rules** remain unchanged.
