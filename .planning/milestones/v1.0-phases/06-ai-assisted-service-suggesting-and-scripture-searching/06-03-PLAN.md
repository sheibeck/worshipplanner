**4. Computed helpers (in script setup):**
```typescript
const hasSermonContext = computed(() => {
  return !!(props.sermonTopic?.trim() || props.sermonPassage)
})

const canAiSearch = computed(() => {
  return !!(aiQuery.value.trim() || hasSermonContext.value)
})
```

**5. AI functions (in script setup):**

`onAiSearch()` — called by search button or Enter key:
- Sets `aiLoading = true`, `aiError = false`, `aiResults = []`.
- Calls `getScriptureSuggestions({ sermonTopic: props.sermonTopic ?? '', sermonPassage: props.sermonPassage, naturalLanguageQuery: aiQuery.value, recentScriptures: props.recentScriptures ?? [] })`.
- On success (non-null): sets `aiResults.value = result`.
- On null: sets `aiError.value = true`.
- Sets `aiLoading = false` in finally.

`onAiSuggest()` — called by "Suggest scripture" button (uses sermon context, no query):
- Sets `aiQuery.value = ''`, then calls `onAiSearch()`.

`onAiRetry()` — clears error, re-triggers search:
- Sets `aiError.value = false`, calls `onAiSearch()`.

`onSelectAiScripture(result: AiScriptureSuggestion)` — selects the AI scripture into the structured fields:
- Sets `localBook.value = result.book`, `localChapter.value = result.chapter`, `localVerseStart.value = result.verseStart`, `localVerseEnd.value = result.verseEnd`.
- Calls `onFieldChange()` to emit update.
- Clears `aiResults.value = []` and `aiQuery.value = ''`.

`aiResultOverlapsSermon(result: AiScriptureSuggestion): boolean`:
- If `props.sermonPassage` is null, return false.
- Build a ScriptureRef from result: `{ book: result.book, chapter: result.chapter, verseStart: result.verseStart, verseEnd: result.verseEnd }`.
- Return `scripturesOverlap(ref, props.sermonPassage)`.
